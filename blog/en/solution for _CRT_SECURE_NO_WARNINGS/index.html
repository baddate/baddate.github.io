Problem: An error about _CRT_SECURE_NO_WARNINGS when I use the std::localtime() function for getting the current systime.

Solution:

  1. Use an alternative feature function in C++.

  2. add this line at the first line.

#define _CRT_SECURE_NO_WARNINGS

  1. add _CRT_SECURE_NO_WARNINGS on the pre-processor(project>properties>C/C++>preprocessor)
