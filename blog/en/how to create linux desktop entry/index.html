    🔖Tips 🔖Linux

  1. create a file named yourappname.desktop in /usr/share/applications/ directory or ~/.local/share/applications/.

  2. input the following lines:

[Desktop Entry]
Type=Application
Terminal=false
Exec=/path/to/app
Name=app
Comment=app comment
Icon=/path/to/app/icon
